Features

Everything in one file

No installation. No framework. Just one Python script with zero mandatory dependencies.

🧩
Jinja2 Template System
Generate .j2 scaffold files per profile, render them into full plugins, validate templates for errors and unknown variables. Regex fallback when Jinja2 isn't installed.
🔐
SHA-256 Integrity
Every generated file is hashed with SHA-256 + MD5 into integrity.json. --verify-integrity detects any post-generation tampering instantly.
✅
10-Area Validator
Full plugin validator covering manifest, hooks, folder structure, UI files, config, icons, libs, and metadata. Auto-runs after generation.
🎨
5 SVG Icon Variants
Every plugin gets 5 icon variants: dark · neon · pixel · symbolic · gradient — each with unique geometry and decorations.
🖥
Bilingual Dark GUI
6-tab Tkinter GUI with deep dark theme, Consolas font, auto-sizing window, and PL/EN language toggle. No scrolling About tab — everything centred.
⌨️
Full CLI (8 Commands)
Fully scriptable. Works headless — no display or tkinter needed. --json flag for CI/CD pipelines. Integrates as a library.
🔌
Meta-Plugin System
Extend the generator without touching core code. Drop .py files into generator_plugins/ — each exposes apply(staged, context).
🧪
Pytest Suite + Selftest
4 pytest files per plugin: test_load · test_commands · test_ui · test_config. Plus a standalone selftest.py requiring no pytest install.
📦
4 UI Panel Layouts
Generated panel CSS uses JetBrains Mono, dot-grid background, glassmorphism modal, 4 px custom scrollbar. Layouts: sidebar · tabs · modal · split.

GUI

6-tab bilingual interface

Deep dark theme, Consolas font, auto-sizing window, PL/EN toggle. All content fits without scrolling.

✦ Generator
Full plugin creation form: name, ID, author, license, description, profile selector (10 options), icon style, code style, UI layout, output directory. One-click generate + ZIP build.
◆ Wizard
4-step interactive creation assistant: basic info → profile & style → hooks & permissions → confirm & generate. Step indicator bar with highlighted active step.
◈ Templates
Generate .j2 scaffold files, validate templates, test-render a plugin from a template. Output area shows file list and validation report.
⚙ Options
Toggle 12 optional output files: pytest tests, selftest.py, CHANGELOG, requirements, install scripts, diagnostics, integrity.json, update.json, plugin-info.json, meta-plugins, auto-validate.
✔ Validator
Two sections: validate a manifest.yaml file, or validate an entire plugin folder (10 areas). SHA-256 integrity verify button. Shared output log.
◈ About
Redesigned in v4.3.0: centred label grid with logo, version badge, author info table, copyright. No scrolling — everything always visible. Language-aware key labels.
